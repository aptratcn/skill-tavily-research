# Tavily Research - 深度研究助手

> 把Tavily搜索变成结构化研究报告

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 快速开始

```bash
# 克隆
git clone https://github.com/aptratcn/skill-tavily-research.git
cd skill-tavily-research

# 设置API Key
export TAVILY_API_KEY="tvly-..."

# 搜索并生成报告
node scripts/tavily-search.mjs --query "AI agent framework" --depth advanced
```

## 功能

- 🔍 Tavily API集成
- 📊 自动格式化研究报告
- 🏷️ 符合content-creator命名规范
- 💾 一键保存到指定目录

## 依赖

- Node.js 18+
- Tavily API Key
